# MyBook
这是一个可以查看各类图书以及书评的Android APP，支持分类浏览、图书搜索、查看详情、查看书评等功能，。后端采用php+mysql做数据支持，数据来源于豆瓣API。


## 技术要点：
本应用的技术要点有：
- Fragment+viewpager+tablayout架构
- 下拉刷新和上拉加载的实现
- image-loader图片加载框架
- android-async-http网络加载框架
